# Archivos en el repositorio:

  -  PINNvsKAN_calor.py
    Comparación de arquitecturas PINN y KAN para resolver el problema de control con la ecuación de calor usando como referencia el control óptimo HUM.
  - Parteo_calor.py
    Test de frontera de pareto para la comparación de distintos tamaños de las arquitecturas PINN y fKAN respecto al residuo físico obtenido y respecto al error respeco al control ópimo HUM.
  - PINNvsKAN.py
    Comparación del residuo físico obtenido por arquitecturas PINN y fKAN en el problema de control de calor+ término no lineal.
  - Paretoa7.py
    Test de frontera de pareto para distintos tamaños de las arquitecturas PINN, fKAN y cKAN respecto al residuo físico obtenido al resolver el problema de control de la ecuación del calor más el tñermino no lineal
